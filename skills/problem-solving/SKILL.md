# Problem-Solving Skill

## 概述

问题解答技能，提供步骤化的问题解决方案和解题思路指导。

## 功能

### 1. 步骤化解题

- 将复杂问题分解为小步骤
- 每步都有清晰的目标
- 提供检查点供用户确认
- 最终汇总得出答案

### 2. 多种解题方法

- 提供不同解题思路
- 比较各方法的优劣
- 帮助选择最适合的方法
- 培养灵活思维

### 3. 错误诊断

- 识别用户解题中的错误
- 分析错误原因
- 提供针对性指导
- 避免类似错误

### 4. 思维可视化

- 流程图展示解题步骤
- 逻辑树分析问题结构
- 表格对比分析
- 公式推导过程

## 使用场景

```
User: "解方程 2x + 5 = 13"
AI: "让我们一步步来解这个方程：

     **第一步：理解问题**
     我们需要找到 x 的值，使得 2x + 5 = 13 成立。

     **第二步：移项**
     把 +5 移到等号右边，变成 -5
     2x = 13 - 5
     2x = 8

     **第三步：求解 x**
     两边同时除以 2
     x = 8 ÷ 2
     x = 4

     **验证：**
     2 × 4 + 5 = 8 + 5 = 13 ✓

     你想试试类似的题目吗？"
```
