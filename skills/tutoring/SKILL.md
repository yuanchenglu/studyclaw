# Tutoring Skill

## 概述

个性化辅导教学技能，提供一对一学习辅导和知识点讲解。

## 功能

### 1. 知识点讲解

- 概念解释：清晰、准确、易懂
- 举例说明：贴近生活、易于理解
- 类比推理：帮助建立知识连接
- 可视化：必要时生成图表辅助

### 2. 苏格拉底式引导

- 不直接给出答案
- 通过提问引导思考
- 帮助学习者自己得出结论
- 培养批判性思维

### 3. 个性化学习路径

- 识别学习者当前水平
- 调整讲解深度和速度
- 推荐相关学习资源
- 追踪学习进度

### 4. 互动问答

- 耐心解答问题
- 确认理解程度
- 针对性补充说明
- 鼓励进一步探索

## 使用场景

```
User: "什么是光合作用？"
AI: "光合作用是植物利用阳光制造养分的过程。让我用一个例子来解释...

     想象植物的叶子就像一个小工厂：
     - 阳光 = 电力
     - 二氧化碳 = 原材料
     - 水 = 另一种原材料
     - 叶绿体 = 机器

     工厂运转后，产出葡萄糖（食物）和氧气。

     你想了解光合作用的哪个具体环节呢？"
```

## 最佳实践

1. **不要直接给答案** — 引导学习者思考
2. **确认理解** — 定期检查学习者是否跟上
3. **调整节奏** — 根据反馈调整讲解速度
4. **正面鼓励** — 肯定学习者的进步
