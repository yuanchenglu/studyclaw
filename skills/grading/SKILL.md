# Grading Skill

## 概述

作业批改技能，提供智能化的作业评估和反馈。

## 功能

### 1. 自动批改

- 识别答案对错
- 部分正确给部分分
- 支持多种题型：
  - 选择题
  - 填空题
  - 计算题
  - 简答题
  - 作文

### 2. 详细反馈

- 指出具体错误位置
- 分析错误原因
- 提供正确解法
- 给出改进建议

### 3. 评分标准

- 支持自定义评分标准
- 多维度评估（准确性、完整性、创新性等）
- 提供评分依据说明
- 公平一致的评分

### 4. 学习建议

- 识别薄弱知识点
- 推荐针对性练习
- 提供学习资源
- 追踪进步趋势

## 使用场景

```
User: [上传作业图片]
AI: "**作业批改报告**

     **总分：85/100**

     ✅ 第1题：正确 (10/10)

     ✅ 第2题：正确 (10/10)

     ⚠️ 第3题：部分正确 (8/10)
     错误：计算过程中符号弄反了
     正确解法：-3 × (-2) = 6，不是 -6

     ❌ 第4题：错误 (0/15)
     原因分析：你混淆了两个概念...
     详细讲解：[展开讲解]

     ✅ 第5题：正确 (15/15)

     **学习建议：**
     建议复习「负数乘法」相关内容，这是你的薄弱点。
     推荐练习：[相关练习链接]"
```

## 配置

```json
{
  "skill": "grading",
  "config": {
    "showCorrectAnswers": true,
    "provideExplanations": true,
    "givePartialCredit": true,
    "generatePracticeProblems": true
  }
}
```

## 评分维度

| 维度   | 权重 | 说明               |
| ------ | ---- | ------------------ |
| 准确性 | 40%  | 答案是否正确       |
| 过程   | 30%  | 解题步骤是否合理   |
| 规范性 | 15%  | 格式、单位是否规范 |
| 创新性 | 15%  | 是否有独特见解     |

## 支持的作业类型

- 数学作业
- 语文作文
- 英语翻译
- 物理实验报告
- 化学方程式
- 代码作业
