# Quiz Skill

## 概述

测验生成技能，提供智能化的题目生成和评估功能。

## 功能

### 1. 题目生成

- 根据知识点自动生成题目
- 多种题型支持：
  - 选择题（单选/多选）
  - 填空题
  - 判断题
  - 简答题
  - 计算题
- 难度自适应
- 避免重复题目

### 2. 自适应测试

- 根据答题表现调整难度
- 项目反应理论 (IRT) 支持
- 精准评估能力水平
- 高效测试流程

### 3. 即时反馈

- 答题后立即反馈
- 解释正确答案
- 分析错误原因
- 提供相关知识链接

### 4. 测试报告

- 正确率统计
- 知识点掌握情况
- 时间分析
- 改进建议

## 使用场景

### 快速测验

```
User: "给我出5道初中数学题"
AI: "**初中数学测验**

     第1题 (选择题)：
     若 x² = 9，则 x 的值是？
     A. 3    B. -3    C. ±3    D. 9

     [等待回答...]

     ✓ 正确！答案是 C. ±3
     解析：x² = 9，则 x = ±√9 = ±3

     第2题 (填空题)：
     若 a + b = 7，a - b = 3，则 ab = __
     ...
     "
```

### 专项练习

```
User: "我想练习英语时态"
AI: "**英语时态专项练习**

     根据你的学习记录，建议重点练习：现在完成时

     第1题：
     I _____ (finish) my homework already.

     A. finish    B. finished
     C. have finished    D. had finished

     [等待回答...]"
```

### 测试报告

```
User: "查看我的测试报告"
AI: "**测试报告**

     📊 **总体表现**
     正确率：78% (39/50)
     用时：23分钟
     排名：前 15%

     📈 **知识点分析**
     ✅ 掌握良好：方程求解、几何证明
     ⚠️ 需要加强：函数图像、概率统计

     📝 **错题回顾**
     1. 第12题 - 概率计算
        你的答案：1/3
        正确答案：1/6
        原因：遗漏了等可能事件的情况数...

     🎯 **学习建议**
     建议重点复习「概率统计」章节，
     推荐练习：[概率专项训练]"
```

## 配置

```json
{
  "skill": "quiz",
  "config": {
    "questionTypes": ["choice", "fill", "judgment"],
    "difficulty": "adaptive",
    "showExplanations": true,
    "timeLimit": null,
    "questionCount": 10
  }
}
```

## 题型模板

### 选择题

```json
{
  "type": "choice",
  "question": "题目内容",
  "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
  "answer": "C",
  "explanation": "答案解析",
  "difficulty": 3,
  "tags": ["知识点1", "知识点2"]
}
```

### 填空题

```json
{
  "type": "fill",
  "question": "题目内容，___ 处需要填写",
  "answer": ["答案1", "答案2"],
  "explanation": "答案解析",
  "difficulty": 2
}
```
